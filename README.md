# 看见傅里叶变换

一个面向大学生的全中文傅里叶变换互动学习网站。课程包含六章，将简洁讲解、公式、实时波形、频谱、声音与即时练习放在同一学习流程中。

## 主要功能

- 六章递进课程：信号、参数、波形合成、傅里叶变换、采样与工程应用。
- 拖动频率、振幅和相位后，时域与频域图实时更新。
- 用户主动点击后可播放声音；浏览器不支持音频时，图形实验仍可继续。
- 每章包含即时练习，学习进度自动保存在当前浏览器。
- 支持桌面和手机布局、键盘操作与减少动态效果设置。

## 本地运行

需要 Node.js 20 或更高版本，以及 pnpm。

```bash
pnpm install
pnpm dev
```

打开命令行显示的本地网址即可开始学习。

## 测试与构建

```bash
pnpm test
pnpm build
```

生产构建会生成在 `dist/` 目录中。构建使用相对资源路径，可部署到普通静态网站服务。

## 音频说明

现代浏览器要求用户主动操作后才能播放声音。因此网站不会自动播放；请在互动实验中点击“播放声音”。教学使用的低频信号会按比例移动到可听频段，但频率之间的高低关系保持不变。
