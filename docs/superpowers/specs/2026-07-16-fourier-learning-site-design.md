# 傅里叶变换互动学习网站设计说明

## 目标

为大学生制作一个全中文、分章节、可交互的傅里叶变换学习网站。网站同时服务于直观理解、公式学习和工程应用，界面保持简洁，能够在桌面和手机浏览器中使用。

## 核心体验

网站采用“讲解与实验并排”的学习方式。桌面端左侧呈现概念、公式和当前操作提示，右侧显示可实时操作的波形与频谱；手机端按“讲解在前、实验在后”的顺序纵向排列。

学习者调整频率、振幅、相位、采样率或信号组成后，时域信号、频谱、可选声音和解释文字同步更新。每章末尾设置一道即时练习，并自动保存章节进度。

## 学习路线

### 第一章：从声音与波形认识信号

- 区分横轴、纵轴和随时间变化的信号。
- 通过播放单一音调观察波形。
- 调节振幅，理解响度与振幅的关系。

### 第二章：频率、振幅和相位

- 分别调节三个参数并观察正弦波。
- 对比低频和高频的视觉与听觉差异。
- 理解相位改变位置但不改变单一频率的幅度谱。

### 第三章：把复杂波形拆成正弦波

- 叠加两个或三个正弦分量。
- 使用预设方波观察谐波逐步逼近。
- 建立“复杂信号是许多简单频率之和”的直觉。

### 第四章：傅里叶变换与频谱

- 对照时域和频域两种视角。
- 展示离散傅里叶变换公式，并逐项解释符号。
- 通过实时频谱确认各正弦分量的位置和强度。

### 第五章：采样、混叠和离散傅里叶变换

- 调节采样率并观察采样点。
- 用动画展示超过奈奎斯特频率后出现的混叠。
- 解释采样点数、频率分辨率和计算结果之间的关系。

### 第六章：工程应用

- 音频降噪：削弱指定频段并对比处理前后。
- 图像压缩：用简化的一维切片说明保留主要频率分量的思路。
- 总结从信号、频谱到处理结果的完整流程。

## 页面与导航

- 首页简要说明学习目标，并提供“开始学习”和“继续上次进度”。
- 固定但轻量的章节导航显示六章名称和完成状态。
- 章节页保持一致结构：章节标题、学习目标、讲解、互动实验、公式展开区、即时练习、上一章和下一章。
- 不设置登录、云同步、社区、排行榜或管理后台。

## 视觉设计

整体采用“白纸实验室”风格：

- 纯白背景、深灰文字、单一蓝色强调色。
- 时域使用蓝色实线，频域使用深灰柱形。
- 不使用渐变、玻璃效果、重阴影或纯装饰动画。
- 容器以留白和细边框区分，圆角保持克制。
- 公式使用清晰的数学排版；正文优先使用系统中文字体，避免依赖网络字体。
- 动画只用于波形变化、频谱变化和明确的操作反馈。

## 交互设计

- 滑块同时提供可读数值，支持键盘方向键。
- 实验提供播放或暂停、预设信号、重置和显示公式操作。
- 参数改变后立即重新生成信号并更新图形与解释。
- 音频仅在用户主动点击后播放，遵守浏览器自动播放限制。
- 用户启用“减少动态效果”时，取消非必要过渡。
- 所有按钮、滑块和章节入口均具有可见焦点状态和中文无障碍标签。

## 技术架构

项目使用 Vite、React 和 TypeScript 构建为无需后端的单页应用。图形使用 Canvas 绘制，声音使用 Web Audio API，学习进度使用 localStorage 保存。

主要模块：

- `chapters`：保存六章结构化中文内容、预设参数和练习。
- `signal`：生成正弦波、组合波和方波近似数据。
- `fourier`：计算离散傅里叶变换或快速傅里叶变换结果。
- `audio`：根据当前参数管理振荡器、音量和播放状态。
- `visualization`：绘制时域、采样点和频域图。
- `lesson`：组合章节内容、实验状态和练习状态。
- `progress`：读取、校验并保存本地学习进度。

核心数据流：

`参数控件 → 信号生成 → 傅里叶计算 → 波形、频谱、声音与解释更新`

## 状态与异常处理

- 参数在进入计算前限制到各章节定义的有效范围。
- localStorage 不可用或数据损坏时恢复默认进度，不阻止学习。
- Web Audio API 不可用或播放失败时显示中文提示，图形实验继续工作。
- Canvas 尺寸随容器变化；尺寸暂时为零时跳过绘制，避免异常。
- 章节或预设数据缺失时显示可返回首页的错误状态。

## 响应式设计

- 桌面宽屏采用讲解与实验双栏布局。
- 较窄屏幕改为单栏，并将实验紧跟在对应讲解之后。
- 章节导航在手机上变为可展开列表。
- 图表自动适应容器宽度，触控目标不小于 44 像素。

## 测试与验收

### 自动化测试

- 验证单一正弦波的频谱峰值出现在预期频率。
- 验证组合信号出现对应的多个频谱峰值。
- 验证采样率和参数边界处理。
- 验证章节切换、练习反馈和进度保存。
- 验证音频不可用时的降级状态。

### 人工验收

- 在常见桌面和手机尺寸检查布局、文字和图形。
- 只用键盘完成章节导航、参数调整和练习。
- 检查减少动态效果设置。
- 从第一章顺序学习到第六章，确认术语、公式和交互结果一致。

## 完成标准

- 六章均可进入，核心实验可实时交互。
- 公式、图形和中文解释对应一致。
- 学习进度刷新页面后仍能恢复。
- 音频失败不影响其他功能。
- 桌面端和手机端均无横向溢出，主要操作可用键盘完成。
